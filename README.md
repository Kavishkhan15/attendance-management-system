# Cloud-Based Attendance Management System

## Project Overview

The Cloud-Based Attendance Management System is a web application developed using HTML, CSS, and JavaScript. It allows users to manage student attendance records efficiently through a simple and responsive interface.

The application is designed as a cloud computing mini-project and can be deployed using Azure Static Web Apps. Attendance records are stored in the browser's Local Storage, allowing data to persist even after the page is refreshed.

---

## Features

* Add student attendance records
* Record student name and roll number
* Auto-filled attendance date
* Mark students as Present or Absent
* View attendance records in a table
* Search attendance records
* Edit attendance records
* Delete attendance records
* Clear all records
* Attendance statistics dashboard
* Export attendance records to CSV
* Dark/Light mode toggle
* Responsive design for desktop and mobile devices
* Local Storage support for persistent data

## Installation and Usage

### Method 1: Run Locally

1. Download the project files.
2. Place the following files in the same folder:

   * index.html
   * style.css
   * script.js
3. Open index.html in a web browser.

### Method 2: Deploy on Azure

1. Create a GitHub repository.
2. Upload all project files.
3. Create an Azure Static Web App.
4. Connect Azure to GitHub.
5. Deploy the application.
6. Access the generated Azure URL.
